Welcome to Ormi Bot

Your All-in-One Discord Bot

Ormi Bot is designed to enhance your Discord server with powerful moderation tools, fun commands, and useful utilities. Whether you're running a gaming community, study group, or just hanging out with friends, Ormi Bot has everything you need.

Setup Guide

1

Invite the Bot

Add Ormi Bot to your server using the invite link below:

Add to Discord
2

Set Up Logging

Create dedicated channels for mod logs and configure them:

/slc #mod-logs - Set moderation log channel
/sdlc #deleted-msgs - Set deleted messages log
3

Configure Welcome Messages

Set up automatic welcome messages for new members:

/swc #welcome - Set welcome channel
/swm Welcome {user} to {server}! - Set welcome message
4

Customize Auto-roles

Automatically assign roles to new members:

/autorole @Member - Set auto-role for new members

Need help? Join our support server for assistance.

Features Overview

Moderation System

  • Advanced warning system with case management
  • Temporary and permanent bans, mutes, and kicks
  • Bulk message deletion and channel management
  • Detailed logging for all moderation actions

Fun & Entertainment

  • Interactive games like RPS and 8ball
  • Memes, jokes, and fun ratings
  • Social commands like hug and ship
  • Random generators for decisions

Utility Tools

  • Server and user information commands
  • AFK status management
  • Message snipe for recent deletions
  • Voting and polling system

Server Configuration

  • Customizable welcome messages
  • Advanced logging options
  • Auto-role assignment
  • Prefix customization
