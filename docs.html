Welcome to Ormi Bot

Your All-in-One Discord Bot

Ormi Bot is designed to enhance your Discord server with powerful moderation tools, fun commands, and useful utilities. Whether you're running a gaming community, study group, or just hanging out with friends, Ormi Bot has everything you need.

Setup Guide

1

Invite the Bot

Add Ormi Bot to your server using the invite link below:

Add to Discord
2

Set Up Logging

Create dedicated channels for mod logs and configure them:

/slc #mod-logs - Set moderation log channel
/sdlc #deleted-msgs - Set deleted messages log
3

Configure Welcome Messages

Set up automatic welcome messages for new members:

/swc #welcome - Set welcome channel
/swm Welcome {user} to {server}! - Set welcome message
4

Customize Auto-roles

Automatically assign roles to new members:

/autorole @Member - Set auto-role for new members

Need help? Join our support server for assistance.

All Commands

Explore the full range of commands available in Ormi Bot, organized by category.

Information

  • /userinfo [member]

    Shows detailed information about a user.

  • /serverinfo

    Displays comprehensive server statistics and details.

  • /ping

    Shows bot latency and connection quality with animations.

  • /uptime

    Shows how long the bot has been online with cute formatting.

  • /avatar [member]

    Shows user avatar with direct link.

  • /roleinfo

    Shows detailed information about a role.

  • /emojiinfo

    Shows emoji information (ID, creation date).

  • /recentjoins

    Shows recently joined members (last 10).

Moderation

  • /kick [reason]

    Kicks a user from the server with confirmation.

  • /ban [duration] [reason]

    Bans a user from the server permanently or temporarily.

  • /unban

    Unbans a previously banned user.

  • /softban [days] [reason]

    Bans and immediately unbans to delete messages.

  • /mute [duration] [reason]

    Mutes a user for a specified time with timer.

  • /unmute

    Unmutes a user with confirmation.

  • /warn [reason]

    Gives a warning to a user with case ID.

  • /warnings

    Shows a user's warnings with detailed info.

  • /clearwarns

    Clears a user's warnings with confirmation.

  • /delwarn

    Deletes a specific warning with verification.

  • /editcase

    Edits the reason for a specific warning.

  • /case

    Shows details about a specific warning.

  • /purge [user]

    Deletes messages (1-100) with user filter.

  • /slowmode

    Sets slowmode for the channel (0-21600).

  • /lock [reason]

    Locks the channel to prevent sending messages.

  • /unlock [reason]

    Unlocks a previously locked channel.

  • /nick [nickname]

    Changes a user's nickname with preview.

  • /nuke

    Deletes all messages in the channel with safety check.

  • /fg [channel]

    Toggles file and GIF sending permissions for a channel.

  • /voicekick [reason]

    Kicks a user from voice channel.

  • /voiceban [reason]

    Prevents user from joining voice channels.

  • /voiceunban [reason]

    Allows user to join voice channels again.

  • /massban [reason]

    Bans multiple users at once.

  • /hierarchy

    Shows server power hierarchy with visual ranking.

Configuration

  • /prefix

    Changes the bot prefix for this server.

  • /slc

    Sets log channel for moderation actions.

  • /sdlc

    Sets deleted messages log channel.

    Sets welcome channel for new members.

Utility

  • /afk [reason]

    Sets your AFK status. The bot will notify users who mention you.

  • /snipe

    Shows the last deleted message in the current channel.

  • /invite

    Generates an invite link for the bot.

  • /vote

    Creates a poll. Separate options with | (e.g., Option1 | Option2).
