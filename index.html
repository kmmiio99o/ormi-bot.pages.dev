All Commands

Information

  • ;userinfo - Shows user information
  • ;serverinfo - Displays server statistics
  • ;avatar - Shows user avatar
  • ;roleinfo - Shows role information
  • ;ping - Shows bot latency

Fun

  • ;ship - Ship two users together
  • ;8ball - Ask the magic 8-ball
  • ;hug - Send virtual hug
  • ;rate - Rate something 1-10
  • ;rps - Play rock-paper-scissors
  • ;random - Random number generator

Moderation

  • ;kick - Kicks a user
  • ;ban - Bans a user
  • ;unban - Unbans a user
  • ;mute - Mutes a user
  • ;unmute - Unmutes a user
  • ;warn - Warns a user
  • ;warnings - Shows user warnings
  • ;clearwarns - Clears user warnings
  • ;delwarn - Deletes specific warning
  • ;purge - Deletes messages
  • ;slowmode - Sets slowmode
  • ;lock - Locks a channel
  • ;unlock - Unlocks a channel
  • ;nick - Changes nickname
  • ;nuke - Deletes all messages in channel

Music

  • ;play - Plays music from YouTube/Spotify
  • ;skip - Skips current track
  • ;queue - Shows current queue
  • ;np - Shows currently playing track
  • ;stop - Stops playback
  • ;join - Joins voice channel
  • ;leave - Leaves voice channel

Utility

  • ;afk - Sets AFK status
  • ;snipe - Shows last deleted message
  • ;poll - Creates yes/no poll
  • ;invite - Generates invite link
  • ;embed - Creates embed
  • ;vote - Creates poll

Configuration

  • ;slc - Sets log channel
  • ;sdlc - Sets deleted messages log channel
  • ;swc - Sets welcome channel
  • ;swm - Sets welcome message
  • ;say - Sends message as bot
