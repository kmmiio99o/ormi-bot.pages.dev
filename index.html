Bot Commands

Moderation

User Management

  • /kick [reason] - Kicks user
  • /ban [reason] - Bans user
  • /unban - Unbans user
  • /timeout [reason] - Times out user
  • /untimeout - Removes timeout from user

Channel Management

  • /lock - Locks channel
  • /unlock - Unlocks channel
  • /clear [amount=5] - Deletes messages

Fun

  • /meme - Get a random meme
  • /joke - Get a random joke
  • /coinflip - Flip a coin
  • /roll [sides=6] - Roll a dice
  • /8ball - Ask the magic 8ball
  • /ship - Ship two users together
  • /hug - Send virtual hug
  • /slap - Virtual slap action

Utility

  • /ping - Check bot latency
  • /serverinfo - Get server information
  • /userinfo [user] - Get user information
  • /avatar [user] - Get user avatar
  • /help - Show this help message
