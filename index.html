All Commands

Moderation

  • ;kick - Kicks a user from the server
  • ;ban - Bans a user (perm - permanent ban)
  • ;unban - Unbans a user
  • ;mute - Mutes a user for specified time
  • ;unmute - Restores speaking/writing permissions
  • ;warn - Gives a warning to user
  • ;warnings - Shows user warnings
  • ;clearwarns - Clears user warnings
  • ;purge - Clears messages (default: 5)
  • ;slowmode - Sets slowmode (in seconds)
  • ;lock - Locks a channel
  • ;unlock - Unlocks a channel
  • ;nick - Changes user nickname

Music

  • ;play - Plays music from YouTube
  • ;skip - Skips current track
  • ;queue - Shows queue
  • ;np - Shows currently playing track
  • ;join - Joins voice channel
  • ;leave - Leaves voice channel

Information

  • ;userinfo - Shows user information
  • ;serverinfo - Shows server information
  • ;avatar - Shows user avatar
  • ;roleinfo - Shows role information
  • ;ping - Shows bot ping

Utilities

  • ;say - Sends message as bot (admin only)
  • ;embed - Creates embed (admin only)
  • ;invite - Generates invite
  • ;vote - Creates yes/no poll
  • ;random - Generates random number
  • ;8ball - Ask magic 8-ball a question

Configuration

  • ;setlogchannel - Sets log channel
  • ;setdeletedlogchannel - Sets deleted messages log channel
  • ;setwelcomechannel - Sets welcome channel
  • ;setwelcomemessage - Sets welcome message
  • ;setmodrole - Sets the moderator role (server owner only)
  • ;setadminrole - Sets the admin role (server owner only)
