All Commands

ℹ️

Information Commands

User Info

  • ;userinfo - Shows user details
  • ;avatar - Shows user avatar
  • ;roleinfo - Shows role info
  • ;emojiinfo - Displays emoji metadata (ID and creation date)

Server Info

  • ;serverinfo - Server statistics
  • ;ping - Bot latency
  • ;uptime - Shows bot operational duration
  • ;recentjoins - Shows recently joined members (last 10)
🎮

Fun Commands

Games

  • ;8ball - Magic 8-ball
  • ;rps - Rock-paper-scissors
  • ;random - Random number generator
  • ;slap - Virtual slap action

Social

  • ;ship - Ship users
  • ;hug - Virtual hug
  • ;rate - Rate something 1-10
🛡️

Moderation Commands

User Management

  • ;kick - Kick user
  • ;ban - Ban user
  • ;mute - Mute user

Channel Control

  • ;lock - Lock channel
  • ;purge - Delete messages
  • ;slowmode - Set slowmode

Utilities

General

  • ;afk [reason] - Sets AFK status
  • ;snipe - Shows last deleted message
  • ;invite - Generates invite link
  • ;rmdm - Sets a reminder
  • ;qr - Generates QR code from text

Polls & Embeds

  • ;vote - Creates poll
  • ;embed - Creates embed
  • ;translate - Translates text
  • ;lastfm - Shows Last.fm statistics
  • ;ascii/span> - Converts text to ASCII art
⚙️

Configuration

Configuration

  • ;slc - Sets log channel
  • ;sdlc - Sets deleted messages log
  • ;say - Sends message as bot
  • ;autorole - Configures automatic role assignment for new members

Welcome

  • ;swc - Sets welcome channel
  • ;swm - Sets welcome message
  • ;prefix - Modifies bot command prefix for this server
🏷️

Roles Management

Basic

  • ;addrole - Adds role to user
  • ;rmrole - Removes role from user
  • ;createrole - Creates new role

Advanced

  • ;delrole - Deletes role
  • ;editrole - Edits role properties
  • ;rr - Creates reaction role panel

Giveaway

Giveaway

  • ;giveaway - Create a new giveaway
  • ;greroll - Reroll giveaway winner

Management

  • ;endgiveaway - End an active giveaway

Ticket Commands

Setup

  • ;ts - Ticket system setup
  • ;tlog - Set ticket logs channel

Staff Management

  • ;addstaff - Add staff to ticket
  • ;rmstaff - Remove staff from ticket
