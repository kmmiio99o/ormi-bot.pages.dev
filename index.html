Bot Commands

Information

  • /userinfo [member] - Shows detailed information about a user
  • /serverinfo - Displays comprehensive server statistics and details
  • /ping - Shows bot latency and connection quality with animations
  • /uptime - Shows how long the bot has been online with cute formatting
  • /avatar [member] - Shows user avatar with direct link
  • /roleinfo <role> - Shows detailed information about a role
  • /emojiinfo <emoji> - Shows emoji information (ID, creation date)
  • /recentjoins - Shows recently joined members (last 10)

Moderation

  • /kick <member> [reason] - Kicks a user from the server with confirmation
  • /ban <user_id> [duration] [reason] - Bans a user from the server permanently or temporarily
  • /unban <user_id> - Unbans a previously banned user
  • /softban <member> [days] [reason] - Bans and immediately unbans to delete messages
  • /mute <member> [duration] [reason] - Mutes a user for a specified time with timer
  • /unmute <member> - Unmutes a user with confirmation
  • /warn <member> [reason] - Gives a warning to a user with case ID
  • /warnings <member> - Shows a user's warnings with detailed info
  • /clearwarns <member> - Clears a user's warnings with confirmation
  • /delwarn <case_id> - Deletes a specific warning with verification
  • /editcase <case_id> <new_reason> - Edits the reason for a specific warning
  • /case <case_id> - Shows details about a specific warning
  • /purge <amount> [user] - Deletes messages (1-100) with user filter
  • /slowmode <seconds> - Sets slowmode for the channel (0-21600)
  • /lock [reason] - Locks the channel to prevent sending messages
  • /unlock [reason] - Unlocks a previously locked channel
  • /nick <member> [nickname] - Changes a user's nickname with preview
  • /nuke - Deletes all messages in the channel with safety check
  • /fg [channel] - Toggles file and GIF sending permissions for a channel
  • /voicekick <member> [reason] - Kicks a user from voice channel
  • /voiceban <member> [reason] - Prevents user from joining voice channels
  • /voiceunban <member> [reason] - Allows user to join voice channels again
  • /massban <ids> [reason] - Bans multiple users at once
  • /hierarchy - Shows server power hierarchy with visual ranking

Fun

  • /rate <thing> - Rates something 1-10 with animated stars ✨
  • /rps <rock|paper|scissors> - Play rock-paper-scissors with cute animations
  • /8ball <question> - Ask the magic 8-ball a question with mystical responses
  • /ascii <text> - Converts text to beautiful ASCII art
  • /random [min] [max] - Generates random number with cute visualization
  • /ship <user1> <user2> - Ship two users together with compatibility percentage
  • /howgay <user> - Measures gay percentage with fun meter animation
  • /simprate <user> - Checks how much a user simps with cute rating
  • /meme - Get a random meme
  • /joke - Get a random joke
  • /coinflip - Flip a coin
  • /roll [sides=6] - Roll a dice
  • /hug <member> - Send virtual hug
  • /slap <member> - Virtual slap action

Configuration

  • /prefix <prefix> - Changes the bot prefix for this server
  • /slc <channel> - Sets log channel for moderation actions
  • /sdlc <channel> - Sets deleted messages log channel
  • /swc <channel> - Sets welcome channel for new members
  • /swm <message> - Sets welcome message for new members
  • /autorole <role> - Sets role for new members automatically
  • /color <hex> - Shows color sample from HEX code

Utility

  • /afk [reason] - Sets your AFK status. The bot will notify users who mention you.
  • /snipe - Shows the last deleted message in the current channel.
  • /invite - Generates an invite link for the bot.
  • /vote <question> <options> - Creates a poll. Separate options with | (e.g., Option1 | Option2).
